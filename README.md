# Todo App (React JS)

This is a simple Todo Application built using React JS. The application allows users to manage daily tasks efficiently.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Responsive user interface

## Technologies Used
- React JS
- JavaScript
- HTML, CSS

## Future Improvements
- Add local storage support
- Edit task functionality
- Improve UI/UX
